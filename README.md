# 🪶 Sparrow Agent

AI-Powered Resume Analysis Assistant built using Python and Streamlit.

## Features

✅ Upload Resume (PDF)

✅ Extract Resume Text

✅ Analyze Resume Sections

✅ Detect Education, Skills, Projects and Experience

✅ Generate Candidate Summary

✅ Resume Scoring System

✅ Resume Recommendations

## Tech Stack

- Python
- Streamlit
- PyPDF

## Current Version

Resume Analyzer V1
